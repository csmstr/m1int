XML and Web


Instructor(s)

Marc Gaetano

Description

  • 3 ECTS

In this course, students learn key XML technologies (XML, XPath, XSL, XML Schema, RNG, DTD, XQuery, DOM) as well as specific markup languages relevant to website development (XHTML, RSS, RDF, XSL-FO). In addition, the course covers topics such as XML and databases (native XML databases and RDBMS), XML programming APIs (DOM and SAX) or Apache server configuration.
